## Requied

### Node.js
Install Node.js >= v16.13 your way.

### pnpm
Install pnpm.

```
corepack enable
corepack prepare pnpm@latest --activate
```

## Usage

Compile

```
pnpm build
```

Compile each time file saved

```
pnpm dev
```

Lint

```
pnpm lint
```

Lint each time file saved

```
pnpm lint:watch
```

Run

```
pnpm start
```
